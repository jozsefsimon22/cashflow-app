# **App Name**: VizFlow

## Core Features:

- Excel Import: Allow users to upload Excel files adhering to a specific template for cash flow data.
- Invoice Chart: Parse the imported data and display a chart showing unpaid invoices and their due dates over the next 12 weeks. User the document number and name columns from the spreadsheet as tooltips for the chart elements.
- Weekly Summary Table: Generate a table summarizing total receivables and payables for each of the next 12 weeks.

## Style Guidelines:

- Primary color: Strong blue (#2962FF), conveying stability and professionalism.
- Background color: Very light blue (#F0F5FF), providing a clean and unobtrusive backdrop.
- Accent color: Purple (#A362FF) to highlight important data points and interactive elements.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text.
- Use clear and professional icons for file uploads and data visualizations.
- Clean and intuitive layout, focusing on data clarity and easy navigation.
- Subtle animations for loading states and data updates to enhance user experience.
